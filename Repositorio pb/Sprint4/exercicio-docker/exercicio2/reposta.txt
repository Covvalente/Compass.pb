É possível reutilizar containers? Em caso positivo, apresente o comando necessário para reiniciar um dos containers parados em seu ambiente Docker? Não sendo possível reutilizar, justifique sua resposta.

Sim é possível utilizamos para isso os comandos 
docker start <nome_do_container> caso um container estiver parado 

e o comando docker restart <nome_do_container> para quando quisermos um container com configurações diferentes
